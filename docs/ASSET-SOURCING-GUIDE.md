## üîß Post-Generation Processing

### Step 1: Download Images
- Right-click ‚Üí Save image
- Save with temporary name
- Keep organized in folders

### Step 2: Remove Background (if needed)
**Free Tool: remove.bg**
1. Go to: https://www.remove.bg/
2. Upload image
3. Download PNG with transparent background
4. Free tier: 50 images/month

**Alternative: Photopea (Free Photoshop alternative)**
1. Go to: https://www.photopea.com/
2. Open image
3. Use Magic Wand ‚Üí Select background ‚Üí Delete
4. Export as PNG

### Step 3: Resize to Specifications

**Using ImageMagick (command line):**
```bash
# Install ImageMagick
sudo apt-get install imagemagick

# Resize character bodies to 512x512
convert input.png -resize 512x512 -gravity center -extent 512x512 output.png

# Resize backgrounds to 1920x1080
convert input.png -resize 1920x1080^ -gravity center -extent 1920x1080 output.png

# Resize stickers to 200x200
convert input.png -resize 200x200 -gravity center -extent 200x200 output.png
```

**Using Online Tool:**
- https://www.iloveimg.com/resize-image
- Upload, set dimensions, download

### Step 4: Optimize File Size

**Using TinyPNG (Online):**
1. Go to: https://tinypng.com/
2. Upload image (max 20 at once)
3. Download optimized version
4. Free: unlimited

**Using pngquant (Command line):**
```bash
# Install
sudo apt-get install pngquant

# Optimize all PNGs in directory
pngquant --quality=80-95 *.png --ext .png --force

# Batch process
find . -name "*.png" -exec pngquant --quality=80-95 --ext .png --force {} \;
```

### Step 5: Rename Following Convention
```bash
# Example renaming
mv downloaded-shark.png char-body-shark.png
mv crown-icon.png char-acc-crown.png
mv toilet-bg.png bg-toilet.png
```

## ü§ñ Automated Batch Processing Script

Save as `scripts/process-images.sh`:

```bash
#!/bin/bash
# Batch process downloaded images

INPUT_DIR="$1"
OUTPUT_DIR="$2"
SIZE="$3"

if [ -z "$INPUT_DIR" ] || [ -z "$OUTPUT_DIR" ] || [ -z "$SIZE" ]; then
    echo "Usage: ./process-images.sh <input_dir> <output_dir> <size>"
    echo "Example: ./process-images.sh ~/Downloads/characters public/assets/images/characters/bodies 512x512"
    exit 1
fi

mkdir -p "$OUTPUT_DIR"

echo "Processing images from $INPUT_DIR..."

for img in "$INPUT_DIR"/*.{png,jpg,jpeg,PNG,JPG,JPEG} 2>/dev/null; do
    [ -e "$img" ] || continue

    filename=$(basename "$img")
    output="$OUTPUT_DIR/$filename"

    echo "Processing: $filename"

    # Resize and center
    convert "$img" -resize "$SIZE" -gravity center -extent "$SIZE" "$output"

    # Optimize
    pngquant --quality=80-95 "$output" --ext .png --force 2>/dev/null || true
done

echo "‚úÖ Processing complete!"
echo "Output: $OUTPUT_DIR"
```

**Usage:**
```bash
chmod +x scripts/process-images.sh

# Process character bodies
./scripts/process-images.sh ~/Downloads/characters public/assets/images/characters/bodies 512x512

# Process backgrounds
./scripts/process-images.sh ~/Downloads/backgrounds public/assets/images/backgrounds 1920x1080

# Process stickers
./scripts/process-images.sh ~/Downloads/stickers public/assets/images/stickers 200x200
```
